2. Visit a video service on the web (like YouTube) and search for "Screen Reader Demo" and "Mobile Screen Reader" - what are some of the common frustrations (or common support mechanisms) that you see among the examples that are given - how could you fix them (or use them)?

After watching some demos of screen readers I think the main things that could be done better (with JAWS in particular) is the voice. I think with that the software would be much easier to use for the visually impaired if the voice reading to them was much less robotic and much more clear.

3. Devices: Please list two devices (and some of their features) that are made specifically to support Accessibility.

With the last Mac OSX update Mountain Lion, there is the new feature voice dictation. This allows the user to press a key of their choosing twice, speak into the built-in mic, and the computer will type whatever you say. This works absolutely anywhere you are able to type. iOS devices have Siri. Siri is “personal assistant” feature that allows the user to hold the home button on an iOS device and say things like “weather”, “remind me to....” or other general questions and If Siri cannot tell you the answer she will ask you if you would like to search the web for results to your question.

4.Mobile Apps: Please list two apps (either Android or iOS, or both) that are made specifically to support Accessibility.

Facetime for iOS devices is an awesome application that not only lets you talk to a person face to face over the phone, but it also allows the deaf sign to each other. Dragon Dictation is application for the deaf that allows the user to open the application while someone is speaking to them and it will transcribe the speech to text for the user to read.
